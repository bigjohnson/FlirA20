Driver Installation Instruction
===============================

The CD-rom includes drivers for the FLIR Systems firewire interface. It also
contains utility software for displaying a live IR image in a web browser.

1 Requirements
  ------------

The Firewire drivers will run under Microsoft Windows 98 SE, Windows ME, 
Windows 2000 and Windows XP.

For the Firewire interface, the DirectX software package version 8.1 or higher
is required.

The Firewire interface will NOT work on Windows 95, Windows 98 (first edition)
or Windows NT 4.0.

It is highly recommended to keep the operating system up to date with the
latest Service Packs and patches from Microsoft.


2 Installation of the driver software for the Firewire interface
  --------------------------------------------------------------

Please log in as Administrator during this installation.

On Windows ME/2000/XP:

Power up the camera and connect the firewire cable. Insert the firewire cable into a
free port on your PC. Windows will now detect the new hardware and display a New Hardware
Found message together with a dialogue. Insert the CD-ROM into your CD drive. The CD-ROM
will show a window of its own, but just remove that. Follow the wizard guide.

On Windows 98:

Windows 98 doesn’t support Plug and Play for FireWire. 
After you have connected the camera follow the instructions below.

	1 Click Start-Settings-Control Panel in order to start the Control Panel 

	2 Double-click on Add New Hardware

	3 Click Next

	4 Click Next

	5 Select Yes, the device is in the list and select the FLIR ThermaCAM_R3 device.
	  Click Next

	6 Click Finish

	7 Click Reinstall Driver

	8 Click Next

	9 Select Search for a better drive than the one your device is using now
	  Click Next

	10 Check CD_ROM drive. Uncheck all other options.

	11 Click Next

	12 Click Next

	13 Insert Windows 98 CD-ROM if prompted to do so.

	14 If you get version conflict questions, Click Yes.

	15 Click Finish

	16 Click Close

	17 Repeat Step 1 to 16 for the FLIR ThermaCAM device.

	18 Reboot your computer if prompted to do so.


3 Installation of utility software for displaying a live IR image in a web browser
  --------------------------------------------------------------------------------

A web browser needs to be installed first, but this is normally already included in
the operating system.

Two supporting software packages needs to be installed in order to be able to view
live IR images in a web browser. First, install the Java Runtime package (JRE) and
then install the Java Media Framework (JMF).
